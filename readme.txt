Installationsanleitung
1. Bitte kopieren Sie alle Dateien in diesem Verzeichnis in den Ordner Ihres Apache-Servers (htdocs)
2. Importieren Sie die festival.sql-Datei aus dem Verzeichnis "\src\database\festival.sql" in Ihre MariaDB/MySQL-Datenbank
3. Richten Sie gegebenenfalls in der Datei "\init\10_database.php" Ihre Datenbank-Verbindung ein
4. Legen Sie sich auf der Registrierungsseite einen Account an - oder verwenden Sie folgende Test-Login-Daten:

Passwort: testpw123
